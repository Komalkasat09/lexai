# Hindi Temp Workspace

This folder stores Hindi-evaluation temporary assets that were previously in `/tmp`.

## Structure

- `scripts/`: ad-hoc Python utilities used for Hindi eval, summary repair, and translation refinement.
- `logs/`: run logs from Hindi evaluation and summary recompute attempts.

## Notes

- These are operational/debug artifacts and are not part of the core product runtime.
- Keep only files needed for reproducibility; archive or delete stale logs when no longer required.
